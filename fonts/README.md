# フォントについて

## GD-高速道路ゴシックJA（公団ゴシック）

`kodan-gothic.woff2` は、日本の高速道路の案内標識で用いられてきた
**「公団文字（通称 公団ゴシック）」を再現したフリーフォント
「GD-高速道路ゴシックJA」** をこのプロジェクト用にサブセット化したものです。

- **書体名**: GD-高速道路ゴシックJA (GD-HighwayGothicJA), version 1.08b
- **著作権 / 制作**: (C) GroupD.B.P.'s / pumpCurry
- **配布元**: https://542.jp/font/gdhwgoja/
- **ライセンス**: アウトラインデータは **CC BY-SA 4.0**
  （<https://creativecommons.org/licenses/by-sa/4.0/>）

## このリポジトリに含めるにあたっての加工

`kodan-gothic.woff2` はオリジナルの TrueType フォントから、以下の加工を施した
**サブセット版**です（CC BY-SA 4.0 に基づく改変物）。

- `index.html` 内で実際に使用する文字のうち、**フォントに字形が存在する文字のみ**を抽出
  （かな・数字・欧文、および「渋滞・出口・速度・車線・距離」等の標識語彙）
- 字形が空の文字（本フォント未収録の一般漢字）は cmap から除外。これにより、
  それらの文字はブラウザのフォントフォールバックで後続の書体（ヒラギノ／Noto 等）に
  自然に切り替わり、空白になることを防いでいる
- 形式を WOFF2 に変換

オリジナルの完全なフォントおよびライセンス全文は上記の配布元を参照してください。
